# Biub

剖析 MVVM 实现原理，使用 ES6 Proxy 结合观察者模式实现双向绑定

>感谢 [DMQ](https://github.com/DMQ/mvvm) 和 [bowencool](https://github.com/bowencool/bue)，compile 和 subscribe 模块都来自这两位大神
